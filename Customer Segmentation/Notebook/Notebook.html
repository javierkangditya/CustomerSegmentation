Customer Segmentation Analysis (RFM + Clustering)

Javier Kangditya

Introduction

Project Overview

This project analyzes customer transaction behavior and segments customers using the RFM (Recency, Frequency, Monetary) framework combined with clustering techniques.

The objective is to uncover meaningful customer groups to support data-driven marketing and retention strategies.

Unlike predictive modeling tasks, this project focuses on unsupervised learning to identify natural patterns in customer behavior.

Dataset Description

The dataset contains retail transaction records with the following variables:

  • InvoiceNo
  • StockCode
  • Description
  • Quantity
  • InvoiceDate
  • UnitPrice
  • CustomerID
  • Country

This dataset represents real-world transactional behavior suitable for customer segmentation analysis.

Project Goals

  • Clean and prepare transactional data
  • Construct RFM features per customer
  • Perform exploratory data analysis (EDA)
  • Apply clustering to segment customers
  • Interpret customer segments from a business perspective

2. RFM Feature Engineering

# Load library
library(dplyr)
library(lubridate)

# -----------------------------
# 1. Set snapshot date
# -----------------------------
snapshot_date <- max(data$InvoiceDate) + days(1)

# -----------------------------
# 2. Build RFM table
# -----------------------------
rfm <- data %>%
  group_by(CustomerID) %>%
  summarise(
    Recency = as.numeric(snapshot_date - max(InvoiceDate)),
    Frequency = n_distinct(InvoiceNo),
    Monetary = sum(TotalPrice)
  )

Conclusion & Insights

1. Summary of Findings

  • Exploratory Data Analysis (EDA):
    • Customer behavior shows strong skewness in Frequency and Monetary, indicating that a small group of customers contributes most of the revenue.
    • Recency reveals that a significant portion of customers have not purchased recently, indicating potential churn risk.
    • Outliers exist in Monetary, representing high-value customers (VIP segment).
  • RFM Feature Engineering:
    • RFM variables (Recency, Frequency, Monetary) were successfully constructed from transactional data.
    • These features provide a behavioral view of customers based on purchase timing, frequency, and spending value.
  • Clustering Results (K-Means):
    • Customers were segmented into multiple clusters using K-Means clustering.
    • Each cluster shows different behavioral patterns in terms of recency, frequency, and monetary value.
    • A small group of customers represents high-value VIP customers, while the largest group consists of low-value or occasional customers.

2. Practical Implications

  • A small segment of customers contributes disproportionately to revenue, showing a clear Pareto (80/20) effect.
  • VIP customers should be prioritized for retention strategies such as loyalty programs and exclusive offers.
  • Churn-risk customers (high Recency) should be targeted with reactivation campaigns.
  • Occasional customers represent growth potential through targeted promotions and engagement strategies.
  • RFM segmentation provides a simple and effective framework for customer-centric marketing decisions.

3. Recommendations

  • Focus retention strategies on VIP customers to maintain high revenue contribution.
  • Implement win-back campaigns for inactive customers with high Recency values.
  • Use targeted marketing campaigns to increase purchase frequency among occasional customers.
  • Regularly update RFM segmentation to reflect changing customer behavior.
  • Integrate this segmentation approach into CRM systems for data-driven marketing strategies.
